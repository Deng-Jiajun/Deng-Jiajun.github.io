软件详细描述

生活经验告诉我，长期不用的软件，忘了就不会再去用它，看到它也想不起来它是干啥用的了，所以记录一些工具的信息。（PS：软件还是尽量堆到磁贴列表里得好，不太常用的软件，用 Listray 找出来实在算不上优雅，还不如在磁贴列表里找图表来的开心)

fiddler

软件名称：fiddler
获取方式：官网下载（https://www.telerik.com/fiddler）
解决问题：如何为 Windows 10 UWP 应用设置代理，文章描述了UWP无法使用代理（VPN）的原因和解决方案，提及了本工具
使用方法：【WinConfig】 -> 选中需要的UWP应用 -> 【Sava Changes】
应该没什么副作用，直接【Exempt All】完事儿了。

v2rayN

软件名称：v2rayN
获取方式：GitHub下载（https://github.com/2dust/v2rayN/releases）
解决问题：FKwall 的客户端不知道为什么用不了，日志显示”V2ray进程的进程退出码是 4294967295“，给出的解决方法是说因为开了多个客户端，关掉就行，但是并没有，问题一直没有解决。看日志知道用的是V2Rray，那就直接用V2ray好了
使用方法：下载【v2rayN-Core.zip】解压打开 【v2rayN.exe】，填订阅，当SS/SSR用

zPlayer UWP

软件名称：zPlayer UWP
获取方式：微软应用商店
软件用途：视频播放器，就，挺好看的。看测评说比 PotPlayer 占用内存少，我试了下好像没有，反正挺好看的 。我用 PotPlayer

PotPlayer

软件名称：PotPlayer
获取方式：官方下载（http://potplayer.daum.net/?lang=zh_CN）
软件用途：视频播放器

Ditto

软件名称：Ditto
获取方式：微软应用商店
软件用途：剪切板增强

Microsoft Whiteboard

软件名称：Microsoft Whiteboard
获取方式：微软应用商店
软件用途：白板，比 Windows 自带的画图工具好看

QuickLook

软件名称：QuickLook
获取方式：微软应用商店
软件用途：空格预览文件。常驻后台，不看任务栏万年想不起来的工具。刚试了一下，打开 300M 的 PDF 要占用 727M 的内存，快速滑动时峰值1500M

Snipaste

软件名称：Snipaste
获取方式：微软应用商店
软件用途：F1截图/F3贴图。自从QQ更新了贴图工具，没用过这款软件的，直接用QQ应该就足够了

Groupy

软件名称：Groupy
获取方式：官网下载（https://www.stardock.com/products/groupy/）
软件用途：多软件标签整合，类似 Clover、QTTabBar，可以整合资源管理器，但是还可以整合软件，比如 Office、ps、pr什么的（有点像是把任务栏挪到上面标签栏位置的意思）。到Steam买更便宜。

为什么不用 QTTabBar 或者 Clover：

  1. QTTabBar 设置实在过于复杂，而且标签不是在顶部，在中段（路径地址栏的位置），实在不习惯，只要勉强用Clover
  2. Clover 官网的版本有广告很烦，又不能入正，只有捐赠？？破解版有去广告，但是在 win10 下稳定性不行，有bug，容易卡死闪退之类的，在发现 Groupy/ExplorerMax 之前，一直用 Clover

体验良好，可以选择将统一进程的程序默认放在同一组。如所有explorer.exe（资源管理器）同一组，所有excel.exe同一组。

ExplorerMax

软件名称：ExplorerMax
获取方式：官网下载（https://explorermax.drivethelife.com/）
软件用途：任务管理器拓展，类似Clover、QTTabBar，没有 Groupy 多软件同标签栏的功能，本来也不需要，我只需要一个带标签的资源管理器就行了。本来打算入正 Groupy 了，结果看到它，确实还是它更合适。准备入正，看下官网，1PC，$40。……还是破解版吧。香。有钱了一定入正，下次一定！
补充一点：Clover 对 win10 暗色主题适配不好，会反白，ExplorerMax 不会，应该比较新的软件对暗色主题的适配都还ok，Groupy看了介绍的配图，是典型 win10 半透明质感的，肯定也是没问题的。

再次补充：转用groupy，这个软件没有想象中的好用。

NewFileTime

软件名称：NewFileTime
获取方式：官网下载（https://www.softwareok.com/?Download=NewFileTime）
软件用途：批量刷新文件创建时间、修改时间、访问时间

Files UWP

软件名称：Files UWP
获取方式：微软应用商店
软件用途：UWP版的资源管理器。功能不全，但是好看。不能作为默认资源管理器，但是好看。有bug，但是好看。

OneQuick

软件名称：OneQuick
获取方式：微软应用商店
软件用途：屏幕边缘操作、短语替换、案件映射等。我主要是需要用到边缘操作，滚轮切换虚拟桌面和调节音量。

Windows Terminal

软件名称：Windows Terminal
获取方式：微软应用商店
软件用途：命令行终端。

EarTrumpet

软件名称：EarTrumpet
获取方式：微软应用商店
软件用途：分应用音量调节，停靠任务栏，可以代替原音量控制图标。

Twinkle Tray

软件名称：Twinkle Tray
获取方式：微软应用商店
软件用途：分显示器调节亮度，停靠任务栏，方便外接显示器亮度调节。

PowerPlanSwitcher

软件名称：PowerPlanSwitcher
获取方式：微软应用商店
软件用途：电源计划调节器，停靠任务栏，方便切换电源计划。夏天笔记本实在太热，只好关闭睿频（最大处理器状态99%）。

Wise System Monitor

软件名称：Wise System Monitor
获取方式：官方下载（https://www.wisecleaner.com.cn/wise-system-monitor.html）
软件用途：悬浮窗，实时显示 CPU 和内存占用率，硬件温度，可以分程序看网络上下行占用，可以杀应用，悬浮窗可以收到屏幕边缘，太丑，是彩色的，不能放进任务栏，差点意思。

ModernFlayouts

软件名称： ModernFlayouts 获取方式：微软应用商店
软件用途：纯美化用途，修改win10的音量、亮度和飞行模式的弹窗，添加键盘锁定（大小写、数字键）的弹窗为 FluentDesign 风格。跟随系统深浅、主题色变化，可自定义移动弹窗出现位置。

Quicker

软件名称： Quicker
获取方式：官网下载（https://getquicker.net/）
软件用途：快捷动作库，替代一些机械性的工作，可视化编程

WinMerge

软件名称： WinMerge
获取方式：互联网
软件用途：文件比对，缺少使用场景，并未深入使用。

DeepL

软件名称：DeepL
获取方式：官网（https://www.deepl.com/translator）
软件用途：基于机器学习的翻译软件

WPD

软件名称：WPD
获取方式：官网（https://wpd.app/）
软件用途：Windows 隐私控制，关闭一些遥测/用户体验计划/麦克风开关 etc.

OpenArk

软件名称：OpenArk
获取方式：官网（https://github.com/BlackINT3/OpenArk）
软件用途：全局热键、内存占用、逆向实用工具、加密、编码……
